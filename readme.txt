Census_Analysis
1. install python 3.6.4 (idle)/anaconda(jupyter notebook)/ use google colab.
2. set the idle path in environment setting. right click on This PC -> go to Properties -> click on Advance System Settings -> click on Environment variables -> go to System Variables -> select Path and click on Edit -> ;C:\Users\AppData\Local\Programs\Python\Python36\Scripts
using google colab: search google colab in search url. -> login with your gmail account -> select new notebook -> select folder sign on left side and upload census dataset.
3. To install packages manually in cmd with version for analysis: eg. pip install pandas==0.25.3 a. ipython==7.10.2 b. numpy==1.18.0 c. pandas==0.25.3 d. matplotlib e. seaborn f. sklearn
4. There are need to spilt the data in 70-30 ratio.
5. Run all analysis , a. In Idle, press F5 b. In Jupyter notebook/ colab notebook, Shift + Enter / Ctrl + Enter and refresh whole notebook then Ctrl + F9.
